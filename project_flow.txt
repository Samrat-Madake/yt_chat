System Flow

Step 1: Input Handling
Take YouTube URL from user
Validate URL

Step 3: Text Preprocessing
RecursiveCharacterTextSplitter

Using:
Sentence Transformers

Step 5: Store in Vector DB
Using:Qdrant
Stores:
embeddings
metadata (chunk text)

Step 6: User Query Processing
User asks question → system:
Converts question → embedding
Searches similar chunks in Qdrant

Step 7: Retrieval

Step 8: LLM Response Generation
Using:
Groq
Input to LLM:
Context (retrieved chunks) + Question
Output:
Final answer grounded in video content

Step 9: Display Answer
Using:
Streamlit
Shows:
user message
bot response
